TECHNICAL SPECIFICATION — GEOMAPPER GM-450 GNSS RTK RECEIVER

Manufacturer: TerraPrecision Geospatial
Model: GM-450-RTK
FCC ID: 2A4GM-GM450
Date: 2024-08-12

The GM-450 is a multi-constellation GNSS (Global Navigation Satellite System) RTK (Real-Time Kinematic) receiver for surveying, GIS (Geographic Information System) data collection, and machine control applications. It supports PPP-RTK (Precise Point Positioning with RTK) via SSR (State Space Representation) corrections.

GNSS Performance:
- Constellations: GPS (L1/L2/L5), GLONASS (G1/G2/G3), Galileo (E1/E5a/E5b/E6), BeiDou (B1I/B2I/B3I), QZSS, SBAS (WAAS/EGNOS/MSAS)
- Channels: 1408 tracking channels
- RTK Accuracy (H/V): 8mm + 1ppm / 15mm + 1ppm (RMS)
- PPP Convergence: < 1 min with SSR corrections
- RTK Initialization: < 5 sec, reliability > 99.9%
- Update Rate: Up to 100 Hz NMEA (National Marine Electronics Association) output
- Tilt Compensation: IMU-based, accuracy ±2cm @ 30° tilt (no calibration needed)

Communication:
- Internal Radio: UHF (Ultra High Frequency) 410–470 MHz, 2W transmit, range 10 km (LOS)
- Cellular: 4G LTE Cat.4 with fallback to 3G UMTS (Universal Mobile Telecommunications System)
- NTRIP (Networked Transport of RTCM via Internet Protocol): Client + Caster built-in
- Bluetooth: BT 5.0
- Wi-Fi: 802.11 b/g/n (2.4 GHz)
- Ports: USB-C, RS-232 (7-pin LEMO), 1PPS (Pulse Per Second) output

Data & Software:
- Data Formats: RTCM 3.3, RINEX 3.04 (Receiver Independent Exchange Format), NMEA 0183
- Logging: Internal 64GB eMMC (embedded MultiMediaCard)
- Software: TerraSurv Office + FieldLink mobile app (Android/iOS)
- Coordinate Systems: 5,000+ predefined datums and projections, custom geoid models (NGS GEOID18)
- Web Portal: Cloud-based with REST API for fleet management and remote configuration

Physical:
- Enclosure: IP68, MIL-STD-810H (drop, shock, vibration)
- Battery: Internal 6,800 mAh Li-ion, 16 hours RTK operation (base), 14 hours (rover)
- Operating Temp: -40°C to +65°C
- Weight: 1.1 kg (with battery, without antenna)
- Antenna: Integrated multi-band choke ring
- Certifications: FCC Part 15/90, IC (Industry Canada), CE RED (Radio Equipment Directive), RoHS
