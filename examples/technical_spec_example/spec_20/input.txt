DATA SHEET — PULSEWAVE PW-900 INDUSTRIAL ETHERNET SWITCH

Manufacturer: NetEdge Industrial
Model: PW-900-MG
Order Code: NEI-PW900-L3M
Date: 2024-01-25

The PW-900 is a managed L3 (Layer 3) industrial Ethernet switch for OT (Operational Technology) networks in manufacturing, oil & gas, and transportation infrastructure. It is designed for TSN (Time-Sensitive Networking) and supports deterministic real-time communication per IEC/IEEE 60802.

Network Specifications:
- Ports: 8x 1GbE RJ45 (PoE+ 30W per port), 4x 10GbE SFP+
- Switching Capacity: 128 Gbps non-blocking
- PoE (Power over Ethernet) Budget: 240W total (IEEE 802.3at)
- TSN Profiles: 802.1Qbv (TAS, Time-Aware Shaper), 802.1Qcc (SRP, Stream Reservation Protocol), 802.1AS (gPTP, generalized Precision Time Protocol)
- Latency: < 5 µs port-to-port (cut-through)
- Redundancy: MRP (Media Redundancy Protocol) per IEC 62439-2, HSR (High-availability Seamless Redundancy), PRP (Parallel Redundancy Protocol)
- VLAN: 4096 VLANs (Virtual LANs), Q-in-Q (802.1ad) stacking
- Security: 802.1X (Port-Based NAC, Network Access Control), MACsec (IEEE 802.1AE), ACL (Access Control List) with L2/L3/L4 filtering
- Routing: Static, OSPF, RIP (Routing Information Protocol), VRRP (Virtual Router Redundancy Protocol)

Industrial Features:
- Operating Temperature: -40°C to +75°C (per IEC 61850-3)
- EMC: IEC 61000-6-2 (industrial immunity), IEC 61000-6-4 (emissions)
- Shock/Vibration: IEC 60068-2-6 (vibration), IEC 60068-2-27 (shock)
- Input Power: 24/48 VDC redundant input, 12–57 VDC wide range
- Power Consumption: 35W (without PoE loads)
- Mounting: DIN rail (TS 35) or 19" rack with adapter
- Enclosure: IP40, aluminum housing, fanless
- Certifications: ATEX/IECEx Zone 2, Class I Div 2 (hazardous locations), DNV-GL (maritime), EN 50121-4 (railway)
- Management: Web GUI, CLI (SSH/Telnet), SNMP v1/v2c/v3, NETCONF/YANG, OPC UA
- Diagnostics: SFP DDM (Digital Diagnostics Monitoring), cable diagnostics TDR (Time Domain Reflectometry), RMON (Remote Monitoring)
- MTBF: 500,000 hours (MIL-HDBK-217F)
- Dimensions: 75mm (W) x 160mm (D) x 130mm (H)
- Weight: 1.8 kg
