PRODUCT BRIEF — SENTINEX SNX-4000 CYBERSECURITY APPLIANCE

Vendor: CyberArc Defense
Model: SNX-4000-NGFW
Part Number: CAD-SNX4000-1U
Date: 2024-06-25

The SNX-4000 is an NGFW (Next-Generation Firewall) with integrated IPS (Intrusion Prevention System), WAF (Web Application Firewall), and ZTNA (Zero Trust Network Access) capabilities. It is designed for mid-enterprise perimeter and DMZ (Demilitarized Zone) deployments.

Performance:
- Firewall Throughput: 40 Gbps (stateful FW)
- IPS Throughput: 12 Gbps with all signatures enabled
- SSL/TLS (Secure Sockets Layer / Transport Layer Security) Inspection: 8 Gbps (TLS 1.3, RSA 2048 + ECDHE)
- Concurrent Sessions: 10M (million)
- New Sessions/sec: 400K CPS (Connections Per Second)
- VPN (Virtual Private Network) Throughput: 20 Gbps IPsec (Internet Protocol Security), 5 Gbps SSL VPN
- ZTNA Users: Up to 10,000 concurrent

Threat Intelligence:
- Signature DB: 25,000+ IPS signatures, updated hourly via TIF (Threat Intelligence Feed)
- Sandboxing: Cloud-based APT (Advanced Persistent Threat) analysis with YARA rule support
- URL Filtering: 120+ categories, ETP (Enterprise Threat Protector) integration
- DLP (Data Loss Prevention): Keyword, regex, and ML-based PII (Personally Identifiable Information) detection
- SIEM (Security Information and Event Management): CEF (Common Event Format) and LEEF (Log Event Extended Format) log export

Management:
- GUI: Web-based HTTPS dashboard, RBAC (Role-Based Access Control)
- CLI: SSH, console (RS-232)
- API: RESTful JSON, GraphQL
- Orchestration: Ansible, Terraform, and proprietary SOAR (Security Orchestration, Automation and Response) connector
- HA (High Availability): Active/Passive and Active/Active clustering, sub-second failover
- Interfaces: 8x 10GbE SFP+, 2x 40GbE QSFP+, 2x 1GbE RJ45 (mgmt)
- Form Factor: 1U rackmount
- Power: Dual 650W AC PSU, hot-swappable
- Certifications: CC EAL4+ (Common Criteria), FIPS 140-2 Level 2, ICSA Labs Certified
