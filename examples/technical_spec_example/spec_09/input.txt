PRODUCT SPECIFICATION — GRIDLINK GLI-5000 SOLAR INVERTER

Manufacturer: SunForce Energy Systems
Model Number: GLI-5000-TL3
Firmware: v4.2.1
Release Date: 2024-02-28

The GLI-5000 is a 3-phase TL (Transformerless) string inverter for commercial PV (Photovoltaic) installations up to 500 kWp. It features dual MPPT (Maximum Power Point Tracking) inputs with wide voltage range and supports grid-forming capabilities for microgrid applications.

Electrical Specifications:
- Max DC Input Power: 550 kW
- MPPT Voltage Range: 200V–1000V DC
- Max Input Current per MPPT: 26A x 2 strings
- Max DC Short-Circuit Current: 40A per MPPT
- Rated AC Output: 500 kVA / 500 kW (PF=1)
- AC Voltage: 3x 480V ±10%, WYE (Y) configuration
- THD (Total Harmonic Distortion): < 3% at rated power
- Frequency Range: 47–63 Hz (adjustable)
- Max Efficiency: 98.8%; Euro Efficiency: 98.4%
- MPPT Efficiency: > 99.5%
- Night-time Consumption: < 5W
- Anti-Islanding: Per UL 1741 SA (Supplement A) and IEEE 1547-2018
- Grid Support: LVRT (Low Voltage Ride Through), HVRT (High Voltage Ride Through), reactive power Q(U) and P(f)
- Communication: RS-485, Ethernet, Wi-Fi, Modbus TCP/RTU, SunSpec Alliance compliant
- Monitoring: Built-in DAS (Data Acquisition System), cloud portal via MQTT (Message Queuing Telemetry Transport)
- Protection: AFCI (Arc Fault Circuit Interrupter), GFDI (Ground Fault Detection and Interruption), SPD (Surge Protection Device) Type II
- Enclosure: IP65, NEMA 3R
- Operating Temp: -25°C to +60°C (derated above 50°C)
- Certifications: UL 1741, IEC 62109-1/-2, NEC 690 (National Electrical Code), FCC Part 15 Class B
